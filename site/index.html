워크플로우 리뷰 허브

하위 페이지 동작 워크플로우, 화면으로 확인하세요

각 카드는 실제 앱 플로우 하나를 단계별로 재생하는 목업입니다. 우측 상단 토글로 다크/라이트를 전환하며 리뷰할 수 있고, 선택은 페이지 간 유지됩니다. 좌우 방향키로도 단계를 넘길 수 있습니다.

액센트 1개: 엔티티 핑크 radius: 카드 16 / 입력 10 / 버튼 pill 폰트: Pretendard

01

온보딩 · 가입 워크플로우

시작 → 로그인(Apple/Google/이메일) → 약관 스크롤 동의 → 권한 설명 카드 → 홈 진입. 상태머신 단계 재생.

 02

홈 · Week 뷰

주간 스트립 + 루틴/스케줄/투두 3분류 홈. 오늘의 루틴 스트릭, 스케줄 타임라인, 투두 체크리스트가 한 화면에 정리되는 구조.

 03

컴포즈 · 실시간 파싱 (핵심)

빈 입력 패턴 제안 + 문장 타이핑 중 날짜/시간/사람/장소 토큰 하이라이트 + 카테고리 자동 분류 + 루틴 자동 전환.

 04

이벤트 상세 시트

제출 직후 바텀시트: 제목/시간/서브투두/지도 스니펫/날씨. 하위 투두 추가 동작 포함.

 05

완료 시트 · "새김"

체크 → 완료 시트에서 사진 첨부 + 한 줄 메모. 완료가 기록이 되는 보상 순간의 워크플로우.

 06

공유 카드 · 웹 뷰

공유 링크 발급 → OG 카드 썸네일 → /s/{id} 웹 페이지. shareAlias만 노출되는 프라이버시 정책 확인.

 07

새김 스토리 · 하루가 기록이 되다

완료·사진·날씨가 하루 스토리로 자동 조립되고, 내가 한 줄을 얹을 수 있다.

 08

설정 · 스마트 기능 제어

Apple Intelligence 보강을 옵션으로 켜고 끄는 설정. 모든 처리는 기기 안에서.
